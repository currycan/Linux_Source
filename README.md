这里记录的是一些国内常用的 Linux 软件镜像源的配置文件，方便直接进行软件源的更换

#如何使用
--------
使用 wget 或是 curl 直接进行下载

- 对于 RPM  
	软件源的配置文件在  
	```/etc/yum.repos.d/CentOS-Base.repo```

- 对于 DEB  
	软件源的配置文件在  
	```/etc/apt/sources.list```
